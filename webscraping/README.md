# Web Scraping Task

## Objective

The objective of this project is to extract data from a public website using Python web scraping techniques.

## Tools and Libraries Used

* Python
* Requests
* BeautifulSoup
* Pandas

## Website Used

https://books.toscrape.com/

## Data Collected

* Book Name
* Price

## Process

1. Connected to the website using Requests.
2. Parsed HTML content using BeautifulSoup.
3. Extracted book names and prices.
4. Stored the extracted data in a dataset.
5. Saved the dataset as a CSV file.

## Output Files

* scraper.py
* books_dataset.csv

## Result

Successfully extracted book data from the website and created a structured dataset for analysis.
